## 📖 简介

本项目是一份系统化的 Golang 全栈开发学习资源，以文档站点的形式呈现，基于 VuePress 构建。内容涵盖了从语言基础到生产实践的完整知识链路，帮助开发者快速成长。

## 🗂️ 内容模块

| 模块 | 说明 |
|------|------|
| **Go 核心基础** | 语法基础、并发编程、网络编程 |
| **AI Agent 开发** | 从入门到精通的 Agent 教程（10 篇） |
| **Web 开发** | Gin 框架等主流 Web 框架实战 |
| **云原生** | Docker、Kubernetes 容器化与编排 |
| **通用技能** | Git 版本控制等开发者必备工具 |
| **开源生态** | 标准库、热门开源库与开源项目速览 |

## 📁 目录结构

```
docs/
├── golang/                  # Go 语言
│   ├── core/                # 核心基础（基础语法、并发、网络）
│   ├── web/                 # Web 开发（Gin 等）
│   └── community/           # 开源生态（标准库、开源库、开源项目）
├── ai/                      # AI Agent 开发教程（10 篇）
└── backend-tech/            # 后端技术
    ├── cloud-native/        # 云原生（Docker、Kubernetes）
    └── general/             # 通用技能（Git 等）
```

## 🚀 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev
```
